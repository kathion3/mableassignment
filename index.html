Introduction to Dataset

  • Analyses conducted using HUD's Public Housing Authorities dataset
  • Dataset includes information on >3,819 public housing authorities.
  • According to HUD, there are approximately 1.2 million households living in public housing units, managed by over 3,300 housing agencies
  • Used variables related to Housing Choice Voucher/Section 8 Housing and Public Housing Units to analyze the distribution of units administered by PHAs across the U.S.

    • Empirical CDFs


    • Total Number of Public Housing Units is the number of all units operated by each PHA. Variable in dataset: PHA_TOTAL_UNITS
    • Total Number of HCV Units is the number of all housing choice voucher/Section 8 units operated by each PHA. Variable in dataset: SECTION8_UNITS_CNT
    • Number of Non-HCV Units is the calculated value of non-HCV units operated by each PHA. Variable in dataset: total_ph_units







    Medians

    • The median number of HCV units across all housing authorities is 77
    • The median number of non-HCV public housing units across all housing authorities is 50
    • The median number of total public housing units across all housing authorities is 185
    • Excel sheets, Python/R scripts or code snippets used for analysis, with explanations of how each step was implemented.

    Condensed Dataset

    • This condensed dataset, available for download as CSV, includes essential information and program size data about all PHAs in the HUD dataset

    • SECTION8_UNITS_CNT is the number of HCV units per PHA
    • PHA_TOTAL_UNITS is the total number of units per PHA
    • total_ph_units was calculated to approximate the number of non-HCV units per PHA
      • total_ph_units = PHA_TOTAL_UNITS - SETION8_UNITS_CNT
